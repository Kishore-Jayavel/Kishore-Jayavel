## 🚀 Featured Projects

### 🦯 AI-Powered Indoor Blind Navigation System
An assistive navigation solution designed to help visually impaired people move safely in indoor environments.

- Detects people, chairs, tables and doors using an Edge Impulse object-detection model
- Measures obstacle distance using ultrasonic sensors
- Uses IMU sensors to identify incorrect turns and movement direction
- Provides real-time voice and buzzer alerts
- Achieved **93.8% model accuracy**
- Built using Raspberry Pi, webcam, IoT sensors and Edge AI

### 🎙️ Human vs Machine Voice Classification
An offline AI system that distinguishes human speech from AI-generated voices.

- Extracted MFCC and Mel Spectrogram audio features
- Compared CNN, Random Forest and SVM models
- Developed a Flask-based interface for real-time prediction

### 🏭 ZERA AI — Industrial Safety Agent
An intelligent industrial safety system designed to improve hazardous-energy awareness and decision support.

- Uses AI agents for safety analysis and recommendations
- Built using Google ADK, Gemini and Qdrant
- Focuses on industrial risk identification and safer maintenance decisions

### 💧 Underground Pipeline Leakage Localisation
An AI-assisted system for detecting and estimating leakage zones in underground pipelines.

- Analyses pressure, flow, vibration and temperature readings
- Combines hydraulic rules with machine-learning predictions
- Displays leakage probability, severity and probable location through a dashboard

### 🔎 Competitive Intelligence AI
A multi-agent RAG platform for collecting, retrieving and analysing competitive business information.

- Uses Retrieval-Augmented Generation and vector search
- Built with Python, FastAPI and React
- Supports structured analysis and intelligent decision-making
